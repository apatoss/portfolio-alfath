Overview

As part of my role at BlueAura Technology, I was responsible for maintaining and continuously enhancing an enterprise Human Resource Management System (HRMS). The system manages HR workflows including employee records, attendance, leave management, and internal reporting for corporate clients.

My work spanned both frontend (Angular/TypeScript) and backend (C#/.NET), with direct database work in Microsoft SQL Server via SSMS. I contributed independently to multiple modules from design through to deployment.

This is a commercial enterprise project developed under BlueAura Technology. Source code is confidential. Only work descriptions and outcomes are shared here.

Key Modules & Features

  • Web dashboard with calendar views, attendance widgets, and upcoming meeting highlights
  • Time Management System (TMS) module focused on attendance tracking
  • Automated report generation module using DevExpress reporting tools
  • Leave approval status components for HR and manager dashboards
  • RESTful API integrations between HRMS modules and external services
  • Bug fixes across multiple system modules — improving stability and UX

My Contributions

  • Designed and built HRMS dashboard UI components using Angular and TypeScript
  • Developed the Time Management System (TMS) module independently from scratch
  • Implemented the automated report generation pipeline using DevExpress
  • Debugged and resolved backend issues in C# and ASP.NET services
  • Optimized SQL Server queries to improve response time for employee data queries
  • Implemented and maintained REST API endpoints for cross-module data integration

Impact

  • Improved overall system stability and reduced bug recurrence
  • Reduced employee data query response time through query optimization
  • Automated report generation replaced manual processes, saving HR admin time
  • Delivered multiple independently developed modules within production timelines
