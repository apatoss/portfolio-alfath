Automated Tooth Pain Diagnosis and Dental Referral System

An AI-powered web system that diagnoses tooth pain based on patient-reported symptoms and automatically refers them to the appropriate dental specialist — reducing unnecessary clinic visits and streamlining dental care.

TypeScript React .NET C# ASP.NET Web API SQL Server REST API

Overview

This system is designed to assist patients in identifying the likely cause of their tooth pain through a guided symptom-based questionnaire. Based on the collected inputs, the system uses a diagnosis engine to determine the probable dental condition and automatically refers the patient to the appropriate type of dental specialist.

The goal is to reduce the burden on general dental clinics by ensuring patients are directed to the right specialist from the start — saving time for both patients and practitioners.

This project is currently in active development. Features and screenshots will be updated as progress continues.

Planned Features

  • Symptom-based questionnaire flow for tooth pain assessment
  • Automated diagnosis engine mapping symptoms to probable dental conditions
  • Intelligent referral system directing patients to the appropriate dental specialist
  • Patient record management and appointment tracking
  • Admin dashboard for dental clinic staff and specialists
  • Referral history and report generation for practitioners
