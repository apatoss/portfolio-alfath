Overview

DotDash is a Robotic Process Automation (RPA) system developed and maintained by BlueAura Technology. The system automates repetitive business workflows for enterprise clients, reducing manual effort and improving operational efficiency.

My role involved developing new features, resolving bugs, improving system security, and managing the deployment pipeline — including the Windows-based installer configuration for client environments.

This is a commercial enterprise project developed under BlueAura Technology. Source code is confidential. Only work descriptions and outcomes are shared here.

Key Features Developed

  • Two-Factor Authentication (2FA) for secure user login
  • Forgot password reset workflow with secure token-based verification
  • Backend logic for user account creation, modification, and deactivation
  • Windows-based installer configuration for client-side RPA deployment
  • Bug fixes across multiple RPA modules — improving stability and user experience
  • REST API enhancements for cross-system data integration

My Contributions

  • Designed and implemented Two-Factor Authentication (2FA) from backend to UI
  • Built forgot password flow — email token generation, verification, and reset logic
  • Developed secure backend account management endpoints using .NET C#
  • Resolved critical bugs in RPA system modules, improving system reliability
  • Managed and configured the Windows-based installer for RPA DotDash deployment
  • Worked with SSMS for database troubleshooting and query handling

Impact

  • Strengthened system security with 2FA implementation
  • Improved user account recovery experience through automated password reset
  • Enhanced system reliability and reduced bug count through targeted fixes
  • Enabled reliable client deployment via properly configured Windows installer
