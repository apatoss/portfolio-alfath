Peer-to-Peer Shoe Trading and Retail Platform

A full-featured marketplace platform for buying, selling, and trading sneakers. Supports peer-to-peer user listings alongside direct retail, with product verification, offer negotiation, and secure transaction flows.

TypeScript Angular .NET C# ASP.NET Web API SQL Server REST API

Overview

This platform is built to serve sneaker enthusiasts who want to buy, sell, or trade shoes in a trusted environment. It combines a peer-to-peer marketplace — where users can list their own items — with a retail section offering direct purchases.

The platform includes a product verification layer to ensure listing authenticity, an offer and negotiation system for trades, and a transaction management flow to keep both buyers and sellers protected throughout the process.

This project is currently in active development. Features and screenshots will be updated as progress continues.

Planned Features

  • User listings — sell or trade sneakers with photos, condition, and pricing
  • Direct retail section for brand new stock purchases
  • Product verification system for authenticity checks
  • Offer and counter-offer negotiation between buyers and sellers
  • Secure transaction and order management flow
  • User profiles with trade history and seller ratings
  • Search and filter by brand, size, condition, and price range
  • Admin dashboard for platform moderation and listing management
