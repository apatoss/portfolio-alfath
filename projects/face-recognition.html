AI Face Recognition Attendance System

A real-time automated attendance system powered by face recognition, designed to replace manual attendance processes with a reliable, camera-based solution.

Python OpenCV MediaPipe Flask MySQL Face Recognition

Overview

This project was developed as my Bachelor's final year project at Management & Science University (MSU). The goal was to automate the attendance recording process using real-time face recognition through a webcam, eliminating the need for manual sign-in sheets or card-based systems.

The system detects and recognizes registered faces in real time, automatically logs their attendance into a MySQL database, and provides an admin dashboard where HR personnel can view, manage, and export attendance records.

Key Features

  • Real-time face detection and recognition using webcam input
  • Automated attendance logging with timestamp into MySQL database
  • Admin dashboard to view attendance records by date and employee
  • Employee registration module with face encoding storage
  • CSV export for attendance report generation
  • Duplicate entry prevention — only logs once per session

My Contributions

  • Developed face detection and recognition module using OpenCV and MediaPipe
  • Integrated MySQL database for attendance and employee data storage
  • Built Flask web dashboard for admin/HR to monitor attendance
  • Implemented CSV export functionality for attendance reports
  • Designed the face encoding pipeline for new employee registration
