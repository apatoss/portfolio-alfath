HRMS Mobile Application (Conficore)

A cross-platform Human Resource Management System mobile application for iOS and Android, developed for Conficore. Enables employees to manage attendance, view HR events, and track real-time status from their mobile devices.

Flutter Dart Swift (iOS) .NET C# Docker MSSQL REST API

Overview

This project was developed at BlueAura Technology as part of a commercial HRMS solution for Conficore. The mobile application was built to complement the existing web-based HRMS by extending its capabilities to iOS and Android devices, giving employees and HR managers convenient mobile access to key HR functions.

The backend was built with .NET C# and containerized using Docker, with Microsoft SQL Server (MSSQL) as the database. iOS development was performed on a Mac machine to support Swift and cross-platform deployment.

This is a commercial project developed under BlueAura Technology. Source code is confidential. Only work descriptions and outcomes are shared here.

Key Features

  • Clock In / Clock Out with real-time attendance tracking
  • Interactive dashboard with dynamic, configurable widgets
  • Real-time attendance status display and updates
  • Calendar view showing leave approvals, public holidays, and HR events
  • Cross-platform support — iOS (Swift) and Android (Flutter/Dart)
  • Secure REST API integration between mobile app and backend services
