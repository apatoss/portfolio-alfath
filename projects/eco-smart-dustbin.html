Eco Smart Dustbin IoT

An IoT-powered smart dustbin that automatically opens its lid when an object is detected and monitors fill level, providing status output via an LCD display and Wi-Fi module.

Arduino C++ Wemos D1 R1 Ultrasonic Sensor IR Sensor Servo Motor IoT

Overview

This project was developed as my Diploma final year project at Politeknik Sultan Mizan Zainal Abidin. The aim was to design and build an eco-friendly smart dustbin using IoT components that can detect the presence of a user, automatically open its lid, and report its fill level status.

The system reduces unnecessary physical contact with the bin (promoting hygiene), provides fill-level awareness to avoid overflow, and demonstrates practical embedded systems programming using Arduino and IoT communication.

Key Features

  • Automatic lid opening and closing via Servo Motor when an object is detected
  • Fill-level detection using Ultrasonic Sensor — alerts when bin is nearly full
  • IR Sensor for proximity-based lid trigger
  • LCD display showing current bin status and fill percentage
  • Wi-Fi connectivity via Wemos D1 R1 for remote status monitoring

My Contributions

  • Designed and assembled the full circuit with all electronic components
  • Programmed sensor logic and servo control using Arduino IDE (C++)
  • Implemented fill-level monitoring with threshold-based alerts on LCD
  • Configured Wemos D1 R1 for Wi-Fi communication and status reporting
  • Tested and calibrated sensor accuracy for reliable real-world performance
